RefMD

Headline

Real-time Markdown collaboration built for teams.

Subheadline

Co-edit specs, guides, and release notes in one OSS workspace with instant previews, presence, and secure sharing.

Primary CTA

Get Started on refmd.io · Read Docs · [View on GitHub] (https://github.com/refmdio/refmd)

What You Get

  • Live editor – Monaco-powered Markdown with Yjs presence, Vim mode, scroll sync, and task toggles.
  • Structured navigation – Drag & drop file tree, backlinks, global search, and fast filters.
  • Share & publish – One-click share links, public pages, PDF/Markdown export, Git sync history.
  • Pluggable platform – Extism WASM plugins add UI panes, slash commands, and custom renderers.

Tech Snapshot

Layer Stack
Frontend React 19, Vite, TanStack Router/Query, Tailwind
Backend Rust (Axum, SQLx, Tokio), PostgreSQL
Realtime WebSockets + Yjs
Deploy Docker, Helm charts, S3 or FS storage

Quick Start

  1. docker compose up -d
  2. Open http://localhost:3000, create an account, add your first note.
  3. Connect Git sync or invite teammates when ready.

Use Cases

  • Engineering specs and decision logs
  • Customer support playbooks and release guides
  • Incident timelines with live collaboration
  • Public knowledge bases powered by Markdown

Community & Support

  • Plugins & examples: github.com/refmdio
